# @webwaka/core — WebWaka OS v4

## Project Overview
This is a TypeScript library package providing shared platform primitives for building SaaS applications on the Cloudflare Workers ecosystem. It is the foundational core for the WebWaka vertical suites (Fintech, Transport, Logistics, etc.).

## Tech Stack
- **Language:** TypeScript 5.x (ESM, ES2022 target)
- **Runtime target:** Cloudflare Workers (`@cloudflare/workers-types`)
- **Framework:** Hono (>=4.0.0, peer dependency)
- **Testing:** Vitest (v8 coverage, thresholds: 80% lines/functions/statements, 75% branches)
- **Package Manager:** pnpm
- **Build Tool:** tsc

## Project Structure
```
.github/workflows/
  publish.yml       — CI: test on all PRs; publish to npm on push to main
src/
  core/
    ai/            — AI abstraction engine (vendor-neutral, 3-tier fallback)
    auth/          — JWT auth, CORS, rate-limiting middleware
    billing/       — Platform billing ledger (integer kobo values)
    booking/       — Booking & scheduling engine
    chat/          — Real-time chat & communication
    document/      — Document & contract management
    geolocation/   — Geolocation & mapping engine (OSM/Google/Mapbox)
    kyc/           — KYC/KYB verification engine
    logger/        — Centralized platform logger (Zero Console Logs)
    notifications/ — Unified SMS/Email (Yournotify + Termii, Nigeria-First)
    rbac/          — Role-based access control
  index.ts         — Main re-export entry point
dist/              — Compiled output (generated by tsc)
server.mjs         — Minimal status server for Replit preview (port 5000)
```

## CI/CD
`.github/workflows/publish.yml` runs on every push/PR to `main`:
1. **test job** — installs deps, type-checks, runs tests with coverage
2. **publish job** — builds and publishes to npm (main branch pushes only, requires `NPM_TOKEN` secret)

## Key Design Principles
- **Nigeria & Africa First:** Financial values stored as integer kobo (not floats)
- **Zero Console Logs:** All logging goes through the platform logger (no direct console.* calls in source)
- **Vendor Neutral AI:** Three-tier fallback (Tenant OpenRouter → Platform OpenRouter → Cloudflare AI)
- **Stateless JWT Auth:** No legacy KV/session storage — all auth is stateless JWT (`@webwaka/core/auth`)
